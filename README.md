# 神经网络与深度学习案例与实践实验所需nndl包

本仓库提供邱锡鹏著《神经网络与深度学习》案例与实践实验1-5中所需的nndl包。该包解决了在实验过程中可能遇到的以下问题：

1. 实验中出现`from nndl.dataset import load_data`报错。
2. 出现`No module named nndl`错误。
3. 运行时显示`ModuleNotFoundError: No module named nndl`错误。
4. 尝试下载nndl模块时显示没有该模块。

## 使用方法

1. **导入nndl包**：将nndl包导入到你的项目同级目录中，解压即可。
2. **虚拟环境**：如果你使用的是虚拟环境，可以将nndl包放入虚拟环境的`...\Anaconda\envs\paddle\Lib\site-packages`目录中。

通过以上方法，你可以顺利解决实验中遇到的nndl模块缺失问题，确保实验的顺利进行。
